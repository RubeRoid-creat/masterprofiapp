# Инструкция по установке обновленного приложения

## Проблема: Кнопка MLM не появляется

## Решение

### Изменения в коде:
1. ✅ Упрощен код обработки кнопки MLM в `ProfileFragment.kt`
2. ✅ Удален дублированный код в `MainActivity.kt`
3. ✅ Добавлена явная видимость кнопки в layout (`android:visibility="visible"`)
4. ✅ Версия приложения увеличена до 1.2.0 (versionCode: 12)

### Ручная установка:

1. **Удалите старое приложение с устройства:**
   ```bash
   adb uninstall com.example.bestapp
   ```

2. **Соберите приложение:**
   ```bash
   gradlew.bat :app:clean :app:assembleDebug
   ```

3. **Установите новое приложение:**
   ```bash
   adb install -r app\build\outputs\apk\debug\app-debug.apk
   ```

   Или используйте скрипт:
   ```bash
   .\install-app.bat
   ```

### Через Android Studio:

1. Откройте проект в Android Studio
2. Выберите устройство/эмулятор
3. Нажмите Run (Shift+F10) или кликните на зеленую кнопку ▶
4. Android Studio автоматически соберет, установит и запустит приложение

### Проверка после установки:

1. Откройте приложение
2. Перейдите в раздел **"Профиль"** (иконка профиля в нижнем меню)
3. Прокрутите вниз до карточки **"Статус верификации"**
4. Должна быть видна кнопка **"Моя команда"** под кнопкой "Пройти верификацию"
5. При нажатии на кнопку должен открыться экран MLM структуры

### Если кнопка все еще не появляется:

1. Проверьте, что версия приложения 1.2.0:
   - Настройки → Приложения → Ремонт Техники → О приложении

2. Перезагрузите приложение:
   - Закройте приложение полностью
   - Откройте снова

3. Очистите кэш приложения:
   ```bash
   adb shell pm clear com.example.bestapp
   ```

4. Проверьте логи:
   ```bash
   adb logcat | findstr ProfileFragment
   ```

## Расположение кнопки MLM:

Кнопка находится в файле `fragment_profile.xml`:
- Карточка: "Статус верификации"
- Позиция: После кнопки "Пройти верификацию"
- ID: `btn_mlm`
- Текст: "Моя команда"
