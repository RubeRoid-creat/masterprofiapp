# BestApp Admin Panel

Веб-интерфейс для администрирования платформы BestApp.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## Настройка

Создайте файл `.env` в корне проекта:

```
VITE_API_URL=http://localhost:3000/api
```

## Функции

- **Дашборд** - общая статистика системы
- **Управление заказами** - просмотр, назначение, отмена заказов
- **Управление пользователями** - блокировка/разблокировка пользователей
- **Верификация** - модерация документов мастеров
- **Жалобы** - обработка жалоб пользователей
- **Бэкапы** - управление резервными копиями базы данных

## Вход

Для входа используйте учетные данные администратора.
