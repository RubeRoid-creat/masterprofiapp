# Тестирование входа администратора

## Проверка

1. **Убедитесь, что backend сервер запущен:**
   ```bash
   cd backend
   npm start
   ```
   Сервер должен быть доступен на `http://localhost:3000`

2. **Проверьте URL API в админ-панели:**
   - Файл `.env` должен содержать: `VITE_API_URL=http://localhost:3000/api`
   - Или проверьте `backend/admin-panel/src/api/api.js` - там должен быть правильный URL

3. **Проверьте CORS настройки:**
   - Backend должен разрешать запросы с `http://localhost:5173`
   - Проверьте `backend/server.js` - там должен быть `app.use(cors())`

4. **Данные для входа:**
   - Email: `admin@test.com`
   - Пароль: `admin123`

## Отладка

1. Откройте консоль браузера (F12)
2. Перейдите на вкладку Network
3. Попробуйте войти
4. Проверьте запрос к `/api/auth/login`:
   - Статус должен быть 200 (успех) или 401 (неверный пароль)
   - Если статус 0 или ошибка сети - backend не запущен или CORS проблема

## Быстрая проверка API

Откройте в браузере или через curl:
```
http://localhost:3000/api/auth/login
```

Должен вернуть ошибку 400 (не хватает данных), но не 404 (маршрут не найден).
