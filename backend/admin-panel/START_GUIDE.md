# Как запустить админ-панель

## Запуск админ-панели

```bash
cd backend/admin-panel
npm run dev
```

Админ-панель будет доступна на: **http://localhost:5173**

## Запуск backend сервера

В **отдельном терминале**:

```bash
cd backend
npm start
```

Backend будет доступен на: **http://localhost:3000**

## Важно!

Оба сервера должны быть запущены одновременно:
1. ✅ Backend на порту 3000
2. ✅ Админ-панель на порту 5173
