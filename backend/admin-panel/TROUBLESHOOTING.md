# Решение проблем с админ-панелью

## Проблема: Админ-панель пустая

### Возможные причины и решения:

1. **Backend сервер не запущен**
   - Убедитесь, что backend сервер запущен на `http://localhost:3000`
   - Запустите: `cd backend && npm start`

2. **Проблемы с CORS**
   - Backend должен разрешать запросы с `http://localhost:5173`
   - Проверьте настройки CORS в `backend/server.js`

3. **Неправильный URL API**
   - Проверьте файл `.env` в `backend/admin-panel/`
   - Должно быть: `VITE_API_URL=http://localhost:3000/api`

4. **Проблемы с аутентификацией**
   - Убедитесь, что вы вошли в систему
   - Проверьте наличие токена в localStorage
   - Убедитесь, что у пользователя роль `admin`

5. **Ошибки в консоли браузера**
   - Откройте DevTools (F12)
   - Проверьте вкладку Console на наличие ошибок
   - Проверьте вкладку Network на наличие неудачных запросов

### Проверка подключения:

1. Откройте консоль браузера (F12)
2. Перейдите на вкладку Network
3. Попробуйте загрузить страницу
4. Проверьте, есть ли запросы к API и их статус

### Создание администратора:

Если у вас нет администратора, создайте его через backend:

```bash
cd backend
node scripts/create-admin.js
```

Или через SQL:

```sql
INSERT INTO users (email, password_hash, name, phone, role)
VALUES ('admin@test.com', '$2a$10$...', 'Администратор', '+79991234567', 'admin');
```
