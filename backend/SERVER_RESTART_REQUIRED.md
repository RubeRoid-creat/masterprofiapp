## Сделаны критические изменения в коде сервера!

Файл `backend/routes/orders.js` был изменен для исправления ошибки 403 при создании заказа.

### 🔴 Что нужно сделать НЕМЕДЛЕННО:

1. **Подключитесь к серверу по SSH:**
```bash
ssh root@212.74.227.208
```

2. **Перейдите в директорию backend:**
```bash
cd ~/masterprofiapp/backend
```

3. **Получите последние изменения:**
```bash
git pull
```

4. **Проверьте список процессов PM2:**
```bash
pm2 list
```

5. **Перезапустите сервер:**

Если процесс называется "server":
```bash
pm2 restart server
```

Если процесса нет в списке:
```bash
pm2 start server.js --name "server"
pm2 save
```

6. **Проверьте логи:**
```bash
pm2 logs server --lines 50
```

7. **Убедитесь, что сервер работает:**
```bash
curl http://localhost:3000/api/version
```
